# katas-python-Jose-Enrique-Alonso
Este es el tercer proyecto de Master donde se practica y afianzan los conocimientos adquiridos en Python

He creado para este proyecto, un entorno virtual que permite aislar los archivos. Para este caso no influye ya que es un único proyecto y no necesita librerías individuales por lo que no generaría ningún conflicto, sin embargo lo utilizo para acostumbrarme a que esto puede suceder en el día a día.

Para realizar cada ejercicio, fui poco a poco leyendo el temario con toda la información. Además, he buscado en internet y en la documentación de Python para entender y ver las opciones que existen en cada momento (por ejemplo la librería math con la opción de "math.pi")

Además, en ciertos ejercicios, he dejado observaciones con las posibilidades que habían, criterios propios de desarrollo y demás.

En muchas ocasiones, he tratado de acostumbrarme a trabajar los strings de cierta manera como lo es el utilizar comillas doblres ("") para declaraciones de opciones y las comillas simples ('') para imprimir mensajes.
